# Teaching JS 4 days in a week

## Disciplined and Consistent Learning

### Track Days

1. Console, Variables, Data types, Keywords, Outputs.
2. Template literals, Operators, Conditional statements, alerts and prompt.
3. String and Array methods, Loops and Object literals.
4. Functions, Scopes, this keyword, try & catch, Arrow function.
5. Array methods, Spread and Rest, DOM.
